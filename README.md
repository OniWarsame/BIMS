# BIMS — Biometric Identity Management System

## Run Locally (localhost)

```bash
# 1. Clone the repo
git clone https://github.com/OniWarsame/BIMS.git
cd BIMS

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open your browser at: **http://localhost:8080**
